# Preferred Tools

Use these tools instead of their standard alternatives:

| Tool  | Replaces | Description         |
| ----- | -------- | ------------------- |
| `rg`  | grep     | Fast search         |
| `fd`  | find     | File finder         |
| `bat` | cat      | Syntax highlighting |
| `eza` | ls       | Git-aware listing   |
| `jq`  | -        | JSON processor      |
| `gh`  | git      | GitHub CLI          |

## Tips

- if you use `gh` command, you can pass GitHub credentials via environment variables. See `gh --help` for more details.
